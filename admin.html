사용 가이드

1. 사용자 추가
"사용자 관리" 탭에서 Google 계정 이메일을 입력하고 권한을 부여하세요.

2. 페이지 권한 설정
"페이지 권한" 탭에서 각 페이지에 학년 칩을 클릭하면 즉시 적용됩니다.
처음에는 "메인에서 가져오기"를 눌러 index.html의 모든 링크를 자동 등록하세요.

3. 일괄 설정
예: "/bk/ 폴더 + 1학년·2학년 허용" → 약사법 자료를 1·2학년 전체에 한 번에 부여.

4. 보안 원칙
• 권한 정보는 Firestore Rules로 보호되어 사용자가 자기 권한 외엔 알 수 없습니다.
• 미등록 페이지는 자동 차단 (안전 우선).
• URL 직접 입력 시에도 차단됩니다 (auth-guard.js 적용 페이지 한정).
